# 🧠 마음동행 AI

독거노인 및 정서 지원이 필요한 사용자를 위한 AI 말벗 서비스

## 주요 기능

- 💬 AI 말벗
- 🎤 음성 입력(STT)
- 🔊 음성 출력(TTS)
- 😊 감정 분석
- 📊 감정 변화 그래프
- 🗓 감정 캘린더
- 📄 PDF 감정 보고서

## 기술 스택

- Streamlit
- FastAPI
- Groq Llama 3.3 70B
- Whisper STT
- SQLite
- Plotly
- Render
